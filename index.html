What are Webwidgets?

WebComponents XS deliver a wide range of predefined technologically advanced components that are easily placed and configured without programming skills. From Databinding through to simple Presentation elements, WebComponents will prove a valuable design and development asset.

Why use Webwidgets?

If you have ever build a website then you know writing code is a time consuming task. What if you coulds select and place a object on a page without worring about responsiveness, authentication and browser support!, ok this a concept that already lives out therebut bare in mind that they all have dependencies that need understanding and programminmg skills.
